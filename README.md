# The Swift
Bot built for the Swift Den Discord: https://discord.gg/DzpaehyZyv
Now available for public use.

## Installation
node.js and npm are required.

```
git clone https://github.com/tecdude/TheSwift.git
cd TheSwift
npm install
nano config.json
```
### config.json sturcutr:

`{
"token": "",
"clientId": "",
"guildId": "",
"OPENAI_API_KEY": ""
}`
